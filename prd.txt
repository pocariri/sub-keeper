1. 무료는 구독 10개까지 등록 + 결제일 + 총액 보기
	프리미엄은 무제한 + 갱신 알림 + 카테고리 분석 + 연간 리포트 + 해지 추천
	
2. 구독 하나에 담을 정보
	1) 서비스명
	2) 금액
	3) 결제 주기
	4) 다음 결제일
	5) 카테고리
	6) 결제수단
	7) 메모
	- 카테고리는 드롭다운 형식으로 주는 게 좋을 것 같아
	- 통화 섞이는 것도 좋아(결제일 기준 환율로)
	
3. 핵심화면, 기능 범위
	- 구독 추가 / 수정 / 삭제
	- 대시보드
	- 카테고리별 분석 차트
	- 갱신 임박 목록
	- 푸시 알람이면 좋을 것 같아
	
4. 가격 정책
	- 무료체험기간 없음
	- 월 2,9000 / 연 29,000
	
5. 관리자 페이지
	- 회원 목록
		- 각자의 플랜
		- 구독 상태
	- 매출 추이
	- 해지율 같은 지표
	- 프리미엄 이용률

6. 로그인
	-	소셜 로그인
	- 아이디 / 비밀번호 로그인
	- 둘다 지원

7. 백엔드
	- Baas 사용
	
8. 디자인
	- 강조색은 없어도 될 것 같아
	- 정말 심플하고 미니멀을 추구하는 디자인
